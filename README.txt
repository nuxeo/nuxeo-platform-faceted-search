Nuxeo Platform Faceted Search Addon
===================================

Overview
--------

This addon adds a new way to browse the whole document repository using visual
filters on metadata.
The new tab appears on the left, just after the navigation tree.

How to deploy it
----------------

Copy the nuxeo-platform-faceted-search-api, nuxeo-platform-faceted-search-jsf and nuxeo-platform-faceted-search-dm
jars to your Nuxeo 5.4.
